Estoy preparando Solaris Express / OpenSolaris en mi Dell D600. Se instala fácilmente y si se tiene un mínimo cuidado no fastidia la instalación de otros sistemas que tengamos en el mismo equipo. Ahora mismo mi D600 tiene Ubuntu, Solaris Express y la licencia original de Windows XP. Tenía una partición primaria VFAT de 10 gigas, y es la que he borrado para que Solaris Express ocupe con la suya. La instalación se ha hecho ahí sin problemas, quedando su gestor de arranque (Grub) en la propia partición, de forma que para arrancar Solaris lo hago desde el Grub que ya tenía para Ubuntu y Windows XP.

# uname -a
SunOS ada 5.11 snv_54 i86pc i386 i86pc
# :-)

El siguiente paso ha sido instalar la red inalámbrica (todavía uso la Intel 2100). La versión que instalé (snv_54) no incluía aun el driver, así que he tenido que obtenerlo de la web de chipsets wifi del grupo de desarrollo para portátiles. En cualquier caso, ha sido sencillísimo: obtener el paquete, instalarlo y reiniciar el sistema. También hay que instalar la utilidad wificonfig, pero mi versión ya la incluía.

# gzip -dc ipw-0.5-pkg.tar.gz | tar xvf -
# pkgadd -d ./ipw-0.5 SUNWipw
# init 6

Una vez disponibles el driver y la utilidad wificonfig, se puede configurar la red desde el administrador de redes de GNOME (que es parte del Java Desktop, escritorio que pretende sustituir a CDE) o a mano. A título de curiosidad, configurarlo a mano para una red con cifrado WEP se realiza con los siguientes comandos:

# ifconfig ipw0 plumb
# wificonfig createprofile MiRed essid=MiRed encryption=wep wepkey1=Miclave
# wificonfig -i ipw0 connect MiRed
# ifconfig ipw0 dhcp

Aun tengo leves problemas: el dispositivo se desconecta a veces, pero basta con reactivar la configuración desde el administrador de red de GNOME para que vuelva a funcionar.

Creo que con esto ya tengo mi portátil preparado para el evento del jueves. :-) Por cierto, muy interesante el grupo de desarrollo de portátiles. En los últimos meses, además, he visto importantes avances, como el soporte de nuevos chipsets Wifi.
